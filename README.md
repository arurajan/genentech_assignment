# Genentech Coding Assessment

This repository contains my solutions to the coding assessment. The instructions for navigating this repository are given below: 
1. There are **four** folders in this repository. The folders have been named according to the instructions provided in the assessment document.
2. Folder **question_1_sdtm** contains solutions to Question 1. The folder consists of 3 files:\
   a. **01_create_ds_domain.Rmd**: Rmd file that contains code for creating the SDTM dataset\
   b. **01_create_ds_domain.log**: Log file corresponding to the .Rmd file\
   c. **ds.csv**: The SDTM DS domain dataset created using the .Rmd file, in .csv format\
3. Folder **question_2_adam** contains solutions to Question 2. The folder consists of 3 files:\
   a. **02_create_adsl.Rmd**: Rmd file that contains code for creating the ADaM ADSL dataset\
   b. **02_create_adsl.log**: Log file corresponding to the .Rmd file\
   c. **adsl.csv**: The ADSL dataset created using the .Rmd file, in .csv format\
4. Folder **question_3_tlg** contains solutions to Question 3. The folder consists of 7 files:\
   a. **01_create_ae_summary_table.Rmd**: Rmd file that contains code for creating the summary table of treatment-emergent adverse events (TEAEs)\
   b. **01_create_ae_summary_table.log**: Log file corresponding to the .Rmd file with the same name\
   c. **ae_summary_table.pdf**: The summary table output in .pdf format\
   d. **02_create_visualizations.Rmd**: Rmd file that contains code for creating visualizations\
   e. **02_create_visualizations.log**: Log file corresponding to the .Rmd file with the same name\
   f. **ae_severity_barchart.png**: Barchart of AE severity distribution by treatment, in .png format\
   g. **top10ae_graph.png**: Graph for top 10 most frequent AEs (with 95% CI for incidence rates), in .png format\
5. Folder **question_4_python** contains solutions to Question 4. The folder contains 1 file:\
   a. **01_create_genai_assistant.ipynb**: .ipynb file that contains code for creating GenAI clinical data assistant. This file also contains       a code chunk that runs 3 example queries and prints the results.
